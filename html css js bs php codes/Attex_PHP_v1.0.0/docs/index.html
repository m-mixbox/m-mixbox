Introduction

Attex PHP is admin template created using PHP & Bootstrap v5.2.3. Attex PHP is crafted by following the best industry standards in order to provide speed and scalability. The clean code structure allows you to build any PHP-based web application without any hustle. The theme includes beautiful dashboards, many ready-to-use hand-crafted components, widgets, pages, etc. The components can be used very easily on any page. It is fully responsive and customizable.
